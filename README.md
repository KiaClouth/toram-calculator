### 目录说明

- .husky: git hooks
- prisma: 棱镜数据库入口
- src/app: 项目页面入口
- src/server: 路由处理和 api 相关文件、请求封装
- src/styles: 项目公用样式、动画、主题、变量等
- src/trpc: trpc
- public: 公共静态资源

### 开发概要

- Next.js 14+
